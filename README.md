## 产品定位

Horo AI 是一款面向中小企业 HR 团队的 AI 原生招聘工作台，核心理念是**工作台 + AI 问答双轨并行**：

- **工作台**提供结构化引导，所有招聘资产（职位、简历、候选人画像、面试记录）持久沉淀
- **Horo AI 助手**作为全局加速层，通过自然语言驱动复杂 AI 任务，零学习成本
- **AI 输出自动回写工作台**，对话结果不再停留在窗口中消失，而是持续积累为企业组织资产

## 核心功能模块

### Horo AI 全局助手

登录后默认进入 Horo AI 页面，提供今日招聘动态播报和 6 个快捷任务入口。支持自然语言驱动所有招聘任务，AI 回答内容自动沉淀到对应工作台模块。

### 职位管理 & JD 生成

- 创建职位，AI 根据岗位信息 + 企业知识库自动生成结构化 JD
- 支持多轮追问和手动调整，JD 生成后自动保存至职位库

### 主动简历获取

- **浏览器插件模式**：一键抓取招聘平台候选人信息
- **RPA Agent 模式**：配置自动化任务，批量采集目标平台简历

### 邮箱简历导入

绑定招聘邮箱，自动同步邮件附件中的简历，支持四种邮箱类型：

| 邮箱类型 | 接入方式 | IMAP Host |
|---------|---------|-----------|
| 网易 126 | IMAP + 客户端授权码 | imap.126.com:993 |
| 网易 163 | IMAP + 客户端授权码 | imap.163.com:993 |
| QQ 邮箱 | IMAP + 客户端授权码 | imap.qq.com:993 |
| 公司邮箱 / 其他 | IMAP 通用配置 | 自定义 Host & Port |

导入后支持单份分析或批量分析，分析结果自动进入候选人库。

### 候选人管理 & 画像

- AI 解析简历，生成六维能力评分（技术能力、沟通表达、学习成长、稳定性、文化匹配、综合潜力）
- **九型人格评估**：AI 推断候选人人格类型，提供岗位匹配度、团队文化匹配度和注意事项
- 多候选人对比 Canvas，支持维度雷达图横向比较

### 面试管理

- 创建面试安排，AI 根据候选人画像自动生成差异化追问题目
- 面试记录归档，支持状态流转（待面试 → 进行中 → 已完成）

### 飞书视频录制

- 配置飞书开放平台应用参数（App ID + App Secret）
- 授权飞书账号，创建面试会议并自动同步录制视频
- AI 语音转写生成结构化复盘报告（问答摘要 + 候选人表现评分）

### 背景调查

**学历验证**支持两种方案：
- **方案 A（推荐）**：输入学信网在线验证码，AI 模拟访问学信网获取结果，自动与简历字段 1:1 比对
- **方案 B（快速上线）**：上传学信网 PDF 报告，AI OCR 解析关键字段进行比对

**工作经历核实**：上传劳动合同/离职证明，AI 提取关键字段与简历比对。

**前司联系人搜寻**：基于候选人工作经历，辅助 HR 定向背调。

**发起调查弹窗**：支持从候选人库勾选候选人，自定义调查项目组合。

### Skill Hub

- 官方技能包（7 大分类，60+ 技能）默认加载
- 支持自然语言创建自定义技能包，经测试沙盒验证后发布
- 影子模式：新 Skill 与现有 Skill 并行运行，对比效果后再正式切换

### 企业知识库

- 支持上传 PDF、Word、TXT、网页 URL 等多种格式
- 知识库内容在 JD 生成、简历筛选等环节被 AI 自动引用
- 支持知识库自动演进（基于 HR 修正行为持续优化）

### 数据看板

- 招聘漏斗、AI 效能对比、Skill 使用效果、HR 修正指数
- 支持按职位、时间段、招聘渠道多维筛选

### 积分系统

- 新用户注册赠送 **30 积分**
- 充值套餐：体验包（50积分/¥29.9）、标准包（200积分/¥59）、专业包（500积分/¥99）、企业包（2000积分/¥299）
- 支持微信支付（扫码）和支付宝（网页支付）

## AI Agent 设计思想

Horo AI 采用 **Harness 治理策略**，在 AI 自主执行与人工接管之间取得平衡：

### 四层架构

| 层级 | 名称 | 职责 |
|------|------|------|
| 第一层 | 意图层（Intent Layer） | 用户通过自然语言/按钮/上下文推荐发起任务，零学习成本 |
| 第二层 | 编排执行层（Horo AI） | 理解意图、拆解任务、调度 Worker、可视化执行过程 |
| 第三层 | Skill 插件层（Skill Hub） | 官方 + 自建技能包统一管理，评估标准灵活定制 |
| 第四层 | 知识底座层 | 权限校验、规则约束、企业知识库沉淀、AI 过程治理 |

### Harness 三层约束

1. **Prompt 级约束**：强制 AI 输出结构化证据链（维度分 + 原文引用 + 风险标签），禁止无依据结论
2. **Skill 级约束**：评估维度与权重在 Skill 包中固化，不同企业可定制不同标准
3. **Harness 治理层**：关键决策节点（如最终录用建议）弹出人工确认卡片，修正行为留痕用于校准模型

### 执行链路

```
用户意图
  → Horo AI 编排（理解意图，拆解任务）
  → Harness 治理层（权限校验 + 规则约束）
  → Worker 执行（Prompt 约束 + Skill 约束）
  → 结构化输出（证据链 + 维度分 + 风险标签）
  → 人工接管节点（关键决策弹出确认卡片）
  → 结果写入系统（修正留痕 + 校准日志）
```

## 快速开始

### 环境要求

- Node.js 22+
- pnpm 9+

### 安装依赖

```bash
cd hule-recruitment
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
pnpm build
```

## 版本规划

### V1.0（核心流程版）

**目标**：打通「简历录入 → 筛选 → 面试 → 复盘」完整闭环，可对外演示和内测上线。

**包含模块**：落地页、登录注册、工作台、职位管理 + JD 生成、候选人画像 + 对比、面试管理、Horo AI 助手、邮箱简历导入、飞书视频复盘、背景调查（学历 + 工作经历）、知识库基础版、积分系统

**时间节点**：研发 8 个工作日 → 测试 3 天 → HR 试用

### V2.0（智能增强版）

**目标**：深化 AI 能力，完善 Skill Hub 和数据演进机制。

**包含模块**：Skill Hub 完整流程（自然语言创建 + 影子模式）、数据看板增强、知识库自动演进、Horo AI 上下文推荐、合规过滤、性能优化（虚拟滚动 + 消息队列）

**时间节点**：V1.0 上线后启动，研发 8 个工作日 → 测试 5 天 → HR 试用
